# Encryption & Decryption Tool

## Project Overview

This project is a simple Encryption & Decryption Tool developed using Python and the Cryptography (Fernet) library.

The application allows users to:

- Generate a secure secret key
- Encrypt sensitive messages
- Store encrypted data securely
- Decrypt encrypted messages back to their original form

This project demonstrates the basic concepts of cryptography and secure data protection.

## Installation

### Step 1: Clone Repository

```bash
git clone https://github.com/yourusername/Encryption-Decryption-Tool-Python-Cryptography.git
```

### Step 2: Navigate to Project Folder

```bash
cd Encryption-Decryption-Tool-Python-Cryptography
```

### Step 3: Install Dependencies

```bash
pip install cryptography
```

OR

```bash
pip install -r requirements.txt
```

---

## Running the Project

### Encrypt a Message

```bash
python encrypt.py
```

Example:

```text
Enter message to encrypt:
Hello Cybersecurity
```

Output:

```text
Encrypted Message:
gAAAAABxxxxxxx...
```

---

### Decrypt a Message

```bash
python decrypt.py
```

Output:

```text
Decrypted Message:
Hello Cybersecurity
```
